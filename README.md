# togovar-stanza

[Togovar-Stanza](https://grch38.togovar.org/stanza "Togovar-Stanza")

## Prerequisites

* Node: 20.19.x
* yarn: 1.22+

## Getting Started

Create a new stanza:

```
$ yarn generate [options] [id]
```

Serve the repository locally:

```
$ yarn run
```

Build stanzas for deployment:

```
$ yarn build
```
